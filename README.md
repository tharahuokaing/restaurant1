## в„№пёҸ бһўбҹҶбһ–бһёбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’

бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ“бҹҒбҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҖбҹ’бһ“бһ»бһ„бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһ’бҹ’бһңбһҫбһ‘бҹҶбһ“бһҫбһ”бһҖбһҳбҹ’бһҳбһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳбһ—бҹ„бһҮбһ“бһёбһҷбһҠбҹ’бһӢбһ¶бһ“ **бһ бһҪ бһҖбһ¶бҹҶбһ„** бһ“бҹ…бһҹбһ¶бһҒбһ¶бһҳбһ»бһҒбһҖбҹҶбһ–бһјбһӣбҹ” бһңбһ¶бһ•бҹ’бһҠбҹ„бһҸбһ‘бҹ…бһӣбһҫбһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ“бҹғбһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ (User Interface) бһ“бһ·бһ„бһӣбҹ’бһ”бһҝбһ“бһ“бҹғбһҖбһ¶бһҡбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһўбһҸбһ·бһҗбһ·бһҮбһ“бһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбҹ”

## вңЁ бһҳбһ»бһҒбһ„бһ¶бһҡбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ—

* **Menu Display:** бһ”бһ„бҹ’бһ бһ¶бһүбһ”бһүбҹ’бһҮбһёбһҳбһ»бһҒбһҳбҹ’бһ бһјбһ” бһ“бһ·бһ„бһ—бҹҒбһҹбһҮбҹ’бһҮбҹҲбһҸбһ¶бһҳбһ”бҹ’бһҡбһ—бҹҒбһ‘бһ“бһёбһҳбһҪбһҷбҹ—бһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢбҹ”
* **Responsive Design:** бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҡбһӣбһјбһ“бһӣбһҫбһӮбҹ’бһҡбһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢ (бһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘, Tablet, бһ“бһ·бһ„ Desktop)бҹ”
* **Fast Loading:** бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҠбһҫбһҳбҹ’бһ”бһёбһұбҹ’бһҷбһҠбһҫбһҡбһӣбһҝбһ“бһ”бҹҶбһ•бһ»бһҸбҹ”
* **Easy Navigation:** бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“ бһ“бһ·бһ„бһ‘бһёбһҸбһ¶бҹҶбһ„бһҡбһ”бһҹбҹӢбһ бһ¶бһ„бҹ”

```bash
https://github.com/tharahuokaing/restaurant.git

```


2. **бһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ Folder бһӮбһҳбҹ’бһҡбҹ„бһ„:**


бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ…бһ»бһ…бһ–бһёбһҡбһҠбһ„бһӣбһҫбһҜбһҖбһҹбһ¶бһҡ `index.html` бһҮбһ¶бһҖбһ¶бһҡбһҹбҹ’бһҡбҹҒбһ…бҹ”
